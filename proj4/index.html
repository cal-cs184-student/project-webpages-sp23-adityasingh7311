Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

A shader program generally sits on the GPU to efficiently compute a part of the rendering pipeline. Shaders have defined inputs and outputs. A vertex shader operates and performs computations on each input vertex. It outputs per-vertex properties that can become the input for fragment shaders. Fragment shaders are used, generally speaking, to color the pixels and the output of the fragment shader is the final output color for each pixel. This comes together to create the lighting and material effects. For example, a vertex shader can set vertex properties like the position and normal and the fragment can use that information to calculate the lighting model formula.

Explain the Blinn-Phong shading model in your own words. Show a screenshot of your Blinn-Phong shader outputting only the ambient component, a screen shot only outputting the diffuse component, a screen shot only outputting the specular component, and one using the entire Blinn-Phong model.

The Blinn-Phong shading model consists of ambient, diffuse, and specular light and calculating it allows us to determine the appearance of the objects we render. Ambient is the constant lighting throughout the scene, diffuse is scattered light from a rough surface, and specular is light reflected from a smooth surface. Blinn-Phong uses the vector halfway between the viewing direction and the light source to measure the specular contribution. The closer the halfway vector is to the surface normal, the higher the specular contribution.

In bump mapping the actual object geometry is not changed but the normals are so that visually we have the illusion of the bump geometry. Displacement mapping on the other hand does change the positions of the vertices to adjust the actual geometry. Visually we can see that the sphere and cloth (rectagle) retain their shape in bump mapping. With the displacement render, the surface of the sphere is no longer smooth but has grooves. The cloth also now has jagged bumps.

Bump mapping does not have significant changes with the different parameters. The mesh itself is more coarse but the bump mapping is not really affected. This means this approach is resilient to the mesh coarseness. With displacement however, there is a significant quality difference immediately visible between the different parameters with the lower parameter having bumps that seem to deform the intended shape. Displacement mapping modifies the actual geometry of the vertices and performs better with the higher parameter mesh.
